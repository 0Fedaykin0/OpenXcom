For Windows, open OpenXcom.2008.sln in Visual C++ 2008 or OpenXcom.2010.sln in Visual C++ 2010 (any editions).
Requires SDL, SDL_mixer and SDL_gfx in SDL/include/ and SDL/lib/ subfolders.
 
For Linux, just use "make". Requires libsdl, libsdl-mixer and libsdl-gfx packages.

Put a copy of X-Com: UFO Defense in a DATA subfolder to run it.
